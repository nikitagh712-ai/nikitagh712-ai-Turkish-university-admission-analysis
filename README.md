# Turkish University Admission Analysis

## Project Overview

This project analyzes Turkish university admission data to identify trends, patterns, and insights related to student admissions.

The workflow includes:
- Data combining
- File handling
- Data cleaning
- Exploratory Data Analysis (EDA)
- Visualization
- Insight generation

---

## Objectives

- Combine multiple admission datasets
- Clean and preprocess the data
- Analyze admission patterns
- Identify important trends in university admissions
- Generate meaningful visual insights

---

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
